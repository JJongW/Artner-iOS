# 작업 계획 및 전체 워크플로우 실행

이 커맨드는 Plan을 시작점으로 **전체 파이프라인을 자동 실행**합니다.
Plan은 PHASE 1 Step 1이며, 완료 후 Context → Checklist → Self-Check →
구현 → 품질검토 → 테스트 → 리포트 → Ship 확인 순으로 자동 진행됩니다.

## 실행 방법

`.claude/commands/orchestrate.md`를 읽고 그 지시사항에 따라 전체 파이프라인을 실행합니다.

**핵심**: PHASE 6 이전에 STOP 하지 마세요. 각 단계 완료 즉시 다음 단계로 진행합니다.
