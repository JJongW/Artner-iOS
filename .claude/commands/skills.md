# 스킬 목록 조회

Artner-iOS 프로젝트에서 사용 가능한 스킬 시스템 정보를 표시합니다.

## 실행 순서

1. 스킬 매니페스트를 읽습니다: `.claude/skills/manifest.json`
2. 디스패처 문서를 읽습니다: `.claude/skills/dispatcher.md`
3. 사용자에게 아래 정보를 표 형태로 정리하여 보여줍니다:
   - 카테고리별 스킬 목록 (스킬 ID, 설명, 슬래시 커맨드)
   - 자연어 디스패치 키워드 매핑 요약
   - 복합 의도 처리 패턴 예시
4. 사용자 질문이 있으면 응답합니다: $ARGUMENTS
