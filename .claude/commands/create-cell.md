# 셀 생성

이 커맨드는 Artner-iOS의 `ios.uikit.create_cell` 스킬을 실행합니다.

## 실행 순서

1. 프로젝트 표준 문서를 읽습니다: `.claude/skills/templates/AI_MANUAL.md`
2. 스킬 프롬프트를 읽고 그 지시사항을 정확히 따릅니다: `.claude/skills/prompts/ios.uikit.create_cell.md`
3. 사용자 요청을 적용합니다: $ARGUMENTS
4. 프롬프트가 다른 스킬을 참조하면 `.claude/skills/prompts/`에서 해당 파일을 읽습니다.
5. 프롬프트가 템플릿을 참조하면 `.claude/skills/templates/`에서 읽습니다.
